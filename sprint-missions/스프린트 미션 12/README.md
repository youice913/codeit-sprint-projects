# 🚲 스프린트 미션 12 - 자전거 대여 수요 예측 프로젝트

## 📌 프로젝트 개요

자전거 대여 시스템 데이터를 분석하고, 머신러닝 모델을 활용하여 대여 수요를 예측하는 프로젝트입니다.  
본 프로젝트의 최종 목표는 **RMSLE(Root Mean Squared Logarithmic Error)를 최대한 낮추는 모델**을 개발하는 것입니다.  
분석 결과는 자전거 배치 및 운영 전략 수립에 활용할 수 있습니다.
